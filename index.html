Temperature Converter

Common Temperature Conversions

Unit Equivalent
0°C= 32°F
100°C= 212°F
-40°C= -40°F
0K= -273.15°C
273.15K= 0°C

Comprehensive Temperature Conversion Guide

Our temperature converter tool provides instant conversions between Celsius, Fahrenheit, and Kelvin scales. Perfect for cooking, scientific work, and weather-related calculations.

How to Use the Temperature Converter

  1. Enter the numerical value you want to convert
  2. Select the input scale (e.g., Fahrenheit)
  3. Select the output scale (e.g., Celsius)
  4. The converted value will appear automatically

Common Temperature Conversion Examples

  • Fahrenheit to Celsius: (°F - 32) × 5/9 = °C
  • Celsius to Fahrenheit: (°C × 9/5) + 32 = °F
  • Celsius to Kelvin: °C + 273.15 = K
  • Kelvin to Celsius: K - 273.15 = °C
